# Auth JWT

## 📖  Description

JWT authentication system with node and express

## 🛠️ Features

- User Provides an Email & Password
- Validate Email & Password
- Validate if user with that email doesn't already exist
- Hash the password
- Save user to DB
- Send a JWT
<br/>

## 📡 Technologies used 

- bcrypt
- express
- express-validator
- jsonwebtoken

## Running Locally

Clone the project

```bash
  git init
  git@github.com:GuiOliver27/email-auth-jwt.git
```

Enter the project directory

```bash
  cd auth-jwt
```

Install the dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Node Version

```bash
  v18.16.0
```
